# LQW Quran Academy Website

A premium, fully responsive website for LQW Quran Academy - an online platform for learning Quran with Tajweed.

## Project Structure

This is a **vanilla HTML, CSS, and JavaScript** project with the following structure:

```
/
├── index.html          # Main HTML file with all page sections
├── style.css           # Complete styling with Islamic green/gold theme
├── script.js           # Interactive features and form handling
├── images/             # All image assets
│   ├── logo.jpeg
│   ├── banner1.jpeg
│   ├── banner2.png
│   ├── hero.png
│   └── video-thumbnail.png
└── server/
    └── index.ts        # Simple Express server to serve static files
```

## Running the Project

The project runs on port 5000 by default. The Express server serves all static files (HTML, CSS, JS, images).

To view the website:
1. The server starts automatically via the workflow
2. Access the website through the web preview
3. All assets are served from the root directory
